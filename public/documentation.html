Welcome to the Movie API

This API serves data about top 10 movies. Visit /movies to get the list.

Request URL Response
GET all movies /movies Returns an array of all movies with details.
GET movie by title /movies/:title Returns a movie object (description, genre, director, image URL, featured).
GET genre by name /genres/:name Returns a genre object (name, description).
GET director by name /directors/:name Returns a director object (bio, birth year, death year).
Register new user /users Registers a new user. Requires JSON body with user data.

Remove movie from favorites /users/:username/favorites/:movieID Returns a message confirming the movie was removed.
Deregister user /users/:username Returns a message confirming user email was removed.
