This RESTful API allows users to get movie information, register and manage users, and favorite movies. All responses are in JSON.

GET /movies

Description: Returns a list of all movies.

Method: GET

Response Example:

{
    "_id": "60f1c...",
    "Title": "Inception",
    "Description": "A thief who steals corporate secrets...",
    "Genre": { "Name": "Sci-Fi", "Description": "Mind-bending..." },
    "Director": { "Name": "Christopher Nolan", "Bio": "...", "Birth": "1970-07-30T00:00:00.000Z" },
    "Actors": ["Leonardo DiCaprio"],
    "ImagePath": "/images/inception.jpg",
    "Featured": true
  
      }

GET /movies/:title

Description: Returns data for a single movie by title.

Method: GET

URL Example: /movies/Inception

Response Example:

{
  "_id": "60f1c...",
  "Title": "Inception",
  "Description": "...",
  "Genre": { "Name": "Sci-Fi", "Description": "..." },
  "Director": { "Name": "Christopher Nolan", "Bio": "...", "Birth": "1970-07-30T00:00:00.000Z" }
}

GET /genres/:name

Description: Returns genre info by genre name.

Method: GET

URL Example: /genres/Sci-Fi

Response Example:

{
  "Name": "Sci-Fi",
  "Description": "Mind-bending science fiction..."
}

GET /directors/:name

Description: Returns director info by name.

Method: GET

URL Example: /directors/Christopher%20Nolan

Response Example:

{
  "Name": "Christopher Nolan",
  "Bio": "...",
  "Birth": "1970-07-30T00:00:00.000Z"
}

POST /users

Description: Register a new user.

Method: POST

Request Body:

{
  "Username": "johndoe",
  "Password": "secret",
  "Email": "johndoe@example.com",
  "Birthday": "1990-01-01"
}

Response Example:

{
  "_id": "60f...",
  "Username": "johndoe",
  "Email": "johndoe@example.com",
  "Birthday": "1990-01-01T00:00:00.000Z",
  "FavoriteMovies": []
}

PUT /users/:username

Description: Update a user's profile info.

Method: PUT

URL Example: /users/johndoe

Request Body (any or all fields):

{
  "Username": "johnny",
  "Password": "newpass",
  "Email": "newemail@example.com",
  "Birthday": "1991-01-01"
}

Response Example:

{
  "_id": "60f...",
  "Username": "johnny",
  "Email": "newemail@example.com",
  "Birthday": "1991-01-01T00:00:00.000Z",
  "FavoriteMovies": []
}

POST /users/:username/favorites/:movieID

Description: Add a movie to a user's favorites.

Method: POST

URL Example: /users/johndoe/favorites/60f123abc456

Response Example:

{
  "message": "Movie \"Inception\" added to johndoe's favorites",
  "FavoriteMovies": ["60f123abc456"]
}

DELETE /users/:username/favorites/:movieID

Description: Remove a movie from a user's favorites.

Method: DELETE

URL Example: /users/johndoe/favorites/60f123abc456

Response Example:

{
  "message": "Movie \"Inception\" removed from johndoe's favorites",
  "FavoriteMovies": []
}

DELETE /users/:username

Description: Deregister (delete) a user by username.

Method: DELETE

URL Example: /users/johndoe

Response Example:

{
  "message": "User \"johndoe\" has been deregistered",
  "user": {
    "_id": "60f...",
    "Username": "johndoe",
    "Email": "johndoe@example.com",
    ...
  }
}
