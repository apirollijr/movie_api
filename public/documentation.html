Welcome to the Movie API

This API serves data about top 10 movies. Visit /movies to get the list.

Request URL Response
GET all movies /movies Returns an array of all movies with full details.
GET movie by title /movies/:title Returns a single movie object by title (including genre, director, release year, etc.).
GET genre by name /genres/:name Returns a genre object with its name and description.
GET director by name /directors/:name Returns a director object with bio, birth year, and optional death year.
Register new user /users Creates a new user. Requires a JSON body with username, email, birthday, etc.
Update user info /users/:username Updates user information. Requires a JSON body with new values.
Add movie to favorites /users/:username/favorites/:movieID Returns a confirmation message after adding the movie to the user's favorites array.
Remove movie from favorites /users/:username/favorites/:movieID Returns a confirmation message after removing the movie from the user's favorites array.
Deregister user /users/:username Deletes a user and returns a confirmation message.
